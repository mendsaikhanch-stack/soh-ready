# 🔮 Хойшлуулсан: CoA санхүү/татвар/ил тод тайлан модуль

**Шийдвэр (2026-06-23):** Play Store **launch-ийн ДАРАА** хийнэ. Одоо энергийг
store-д гаргахад зарцуулна. Энэ файл нь дизайн + хийх ёстой засваруудыг хадгална.

## Зорилго (3 тайлан нэг endpoint-оос)
1. **Санхүүгийн тайлан** — Баланс (актив/эх үүсвэр) + Орлого-үр дүн
2. **Татварын тайлан** — ТАХ-ын TT-02(2) формат руу импортлох
3. **"Шилэн СӨХ" тайлан** — оршин суугчдад график, цуглуулалтын хувь

## ⚠️ As-is ажиллахгүй — засах ёстой зүйлс
Анх ирүүлсэн `app/api/coa-reports/route.ts` нь одоогийн схемтэй таарахгүй:

1. **`ledger_lines` хүснэгт байхгүй** — давхар бичилтийн ledger (debit/credit,
   account_code, coa_id) апп-д огт алга. Бүх тайлан үүн дээр тулгуурласан.
2. **`invoices.coa_id` байхгүй** — invoices нь `sokh_id`-тэй. `coa_id` → `sokh_id`.
3. **Status том үсэг** — код `'PENDING'`/`'PAID'`, схем жижиг `'pending'`/`'paid'`
   → авлага/цуглуулалт үргэлж 0 гарна. Заавал засна.
4. **Буруу env** — `process.env.SUPABASE_URL` → `NEXT_PUBLIC_SUPABASE_URL`.
5. **🔒 Auth огт байхгүй** — service-role key, нэвтрэлт шалгахгүй endpoint нь
   дурын СӨХ-ийн санхүү/татварыг ил болгоно. Заавал auth + tenant-scope нэмэх.
6. **Module-scope `createClient(undefined)`** — #4-тэй хослоод build унагаж магадгүй.
   Env-ийг handler дотор уншина.

## Хоёр зам
- **A. Жинхэнэ давхар бичилт** — `ledger_lines` + дансны төлөвлөгөө, гүйлгээ бүрт
  (нэхэмжлэх төлөгдөх, зардал) ledger бичих. Audit/татварт бэлэн ч том ажил.
- **B. Одоогийн датаг адаптчлах** — орлого `invoices`/`payments`-аас, зардал
  `budget_items`-аас, авлага төлөгдөөгүй нэхэмжлэхээс. Давхар бичилтгүй, хурдан,
  одоо ажиллана. **Ихэнх СӨХ-д хангалттай** (audit-grade татвар хэрэгтэй болтол).

## Дансны кодын зураглал (анхны санаа)
| Код | Утга |
|---|---|
| 1010 | Касс/банк (актив, D−C) |
| 1210 | Авлага (төлөгдөөгүй нэхэмжлэх) |
| 3110 | Хуримтлагдсан ашиг |
| 4010 | Үндсэн орлого — гишүүдийн хураамж (татваргүй) |
| 4020 | Бусад орлого — түрээс/алданги (10% татвартай) |
| 5010 | Цалин, НДШ |
| 5020 | Тог/цахилгаан/ус |
| 5030 | Лифт/хог/засвар |

> Татвар: СӨХ зөвхөн **бусад орлого (4020)** дээр 10% төлнө гэсэн таамаг —
> launch дараа татварын мэргэжилтнээр баталгаажуулах.
