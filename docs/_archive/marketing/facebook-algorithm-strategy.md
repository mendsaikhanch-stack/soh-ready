**Зорилго:** Facebook алгоритм яг одоо креаторуудыг хэрхэн "шагнаж байгаатай" уялдсан контентийн бүтэц боловсруулах.

| Сигнал | Жин | Тайлбар |
|--------|-----|---------|
| **Meaningful comments** (харилцан хариу) | 🔥🔥🔥🔥🔥 | Хэн нэг нь сэтгэгдэлд хариу бичих → 2 дахин буст |
| **Dwell time** (постонд зарцуулсан секунд) | 🔥🔥🔥🔥 | Уртлаг пост уншуулж чадвал → алгоритм "үнэ цэнтэй" гэж үзнэ |
| **Reels-first** (босоо видео) | 🔥🔥🔥🔥🔥 | 2025-аас хойш Reels-д хамгийн их reach өгч байна |

| **Original content boost** | 🔥🔥🔥 | Repost биш, өөрөө бүтээсэн → priority |

## 🏗️ Контент бүтэц — 4 давхар "Pyramid"

```
┌─────────────────────────────────────────┐
│  4. CTA + DM trigger     (10 секунд)    │  ← Conversion
├─────────────────────────────────────────┤
│  3. Engagement trigger   (30 секунд)    │  ← Comment fishing
├─────────────────────────────────────────┤
│  2. Value delivery       (60-90 секунд) │  ← Dwell time
├─────────────────────────────────────────┤
│  1. Hook (Scroll-stop)   (2-3 секунд)   │  ← Stop the thumb
└─────────────────────────────────────────┘
```

## 🪝 1-р давхар: HOOK Patterns (эхний 1-2 мөр)

Facebook feed-д "See more"-ийн өмнө харагдах **115 тэмдэгт** л чухал. Эдгээр загварыг ашигла:

| # | Hook загвар | ХОТОЛ-д тохирох жишээ |
|---|-------------|------------------------|
| 1 | **Loss aversion** (юу алдаж байгаа) | "Жилд 12 сая төгрөг алддагаа мэддэггүй СӨХ-нүүд олон." |
| 2 | **Contrarian** (нийтлэг үзлийг эсэргүүцэх) | "СӨХ-ний тэргүүн сайн дурын ажилтан биш. Тийм бодоход л таны мөнгө алдагдаж байгаа." |
| 3 | **Specific number + curiosity gap** | "47 хороолол энэ системд шилжсэний дараа нэг л зүйл өөрчлөгдсөн нь — гишүүд цуглаанд ирдэг болсон." |
| 4 | **Confession / vulnerability** | "Бид анх ХОТОЛ-ыг бүтээхдээ алдаа гаргасан. Тэр нь биднийг 6 сар хойш татсан." |
| 5 | **Direct question** (хариу нэхэх) | "Танай хорооллын СӨХ Excel дээр төсвөө хөтөлдөг үү? Тэгвэл..." |
| 6 | **"I/We tested X with Y"** | "50 СӨХ-ний тэргүүнтэй ярилцлага хийсний дараа 1 зүйл тодорхой болсон..." |

**⚠️ Үл ажиллах hook:**
- "Сайн байна уу гишүүд ээ!" (generic)
- "Шинэ бүтээгдэхүүнээ танилцуулж байна" (sales pitch шиг)
- Emoji-н хэт олон 🚀🔥✨🎉 (алгоритм spam гэж шошголно)

## ⏱️ 2-р давхар: Content Pacing (Dwell time-д зориулсан)

Facebook алгоритм **уншихад 30+ секунд зарцуулах** контентийг хайж байна. Бүтэц:

```
Line 1: HOOK (115 тэмдэгт хүртэл — feed-д харагдана)
[1 хоосон мөр]
Line 2: Open the loop (асуулт эсвэл маргаантай мэдэгдэл)
[1 хоосон мөр]
Line 3-4: Контекст
[1 хоосон мөр]
✅ Bullet 1
✅ Bullet 2
✅ Bullet 3
[1 хоосон мөр]
"P.S." эсвэл "Гайхалтай нь:" гэсэн twist
[1 хоосон мөр]
CTA + сэтгэгдэлд оруулсан өгөөш
```

**Алтан дүрэм:** 1 мөр = 1 санаа. 5-7 үгээр нэг догол мөр.

**Хэмжээ:** 800-1500 тэмдэгт = Facebook-ийн "sweet spot" 2026 онд. (Урт ч биш, богино ч биш.)

## 💬 3-р давхар: Engagement Triggers

Facebook **comments**-ийг хамгийн өндрөөр үнэлдэг. Сэтгэгдэл татах 7 техник:

| Техник | Жишээ caption |
|--------|---------------|
| **Polarizing question** | "Танай хорооллын СӨХ-ний тэргүүн цалинтай юу, сайн дурын юу? Бид тэргүүний цалин өгөх ёстой гэж бодож байна. Та юу гэж бодож байна?" |
| **Two-option poll in comments** | "1 — Excel дээр хөтөлдөг, эсвэл 2 — апп ашигладаг? Доор бичээрэй." |
| **"Tell me without telling me"** | "Танай СӨХ муу ажилладгийг 1 өгүүлбэрээр илэрхийл." |
| **Lead magnet in comments** | "СӨХ-ний жилийн төсвийн загвар PDF-г сэтгэгдэлд тавилаа. 'Хүсэж байна' гэж бичсэн хүн бүрт DM-аар явуулна." |
| **Tag prompt** | "Танай орон сууцанд ингэж асуудалтай нөхрийг таних уу? Тагла." |
| **Story prompt** | "Танай хорооллын СӨХ-нд тохиолдсон хамгийн муухай түүхээ бичээрэй. Хамгийн их сэтгэгдэл аваагүй пост гарна." |
| **Mid-post question** | (Постын дунд) "...за энэ хэсэгт нэг асуулт. Та хэдэн сар СӨХ-ний төлбөрөө хоцолсон бэ?" |

**🎯 Reply strategy:** Эхний 60 минутад орж ирсэн сэтгэгдэл бүрд хариу бичнэ. Энэ нь алгоритмд "энэ пост ярилцлага үүсгэж байна" гэсэн сигнал илгээнэ.

## 🎬 4-р давхар: Format & Distribution Triggers

### Format-ын priority (2026 онд Facebook-ийн өгсөн жин)

```
1. Босоо video (Reel) 9:16    🔥🔥🔥🔥🔥
2. Carousel (3-10 зураг)      🔥🔥🔥🔥
3. Native фото                🔥🔥🔥
4. Text-only пост             🔥🔥
5. Гадны линктэй пост         🔥
```

### Link-ийн арга (link нь reach-ийг алдаг)

**❌ БҮҮ:** Постын ёроолд `hotol.mn/demo`
**✅ ХАМАХ:** "Линк сэтгэгдэлд оруулсан 👇" → дараа нь өөрөө first comment-д линк
→ Алгоритм reach-ийг 40%+ хадгална

### Posting time (Монгол үзэгчдэд)

| Цаг | Үр дүн |
|-----|--------|
| Өглөө 07:30-08:30 | ✅ Ажилдаа явах замдаа уншина |
| Үд 12:00-13:30 | ✅✅ Хамгийн өндөр |
| Орой 20:00-22:00 | ✅✅✅ Хамгийн өндөр (зурвас бичнэ) |
| Долоо хоногийн өдөр | Мягмар, Лхагва, Пүрэв = шилдэг |

### Velocity boost (эхний 90 минутын стратеги)

1. Пост хийсэн даруйдаа **Stories-д share хийх** (өөрийн профайл руу татна)
2. **3 хувийн чат** (СӨХ-ний тэргүүнтэй, оршин суугчтай) рүү линкийг илгээх — "санал хэлээрэй" гэж
3. **Анхны 5 сэтгэгдэлд** өөрөө reply хийнэ
4. **Messenger group** руу broadcast хийхгүй (spam гэж шошголно)

## 🚫 Facebook-ийн "shadow ban" trigger-ууд (зайлсхийх)

- AI-image харагдах байдалтай (3 хурууны зураг, гялалзсан текстур)
- 5+ emoji эхний мөрөнд
- "Subscribe / Follow" гэсэн шууд CTA
- Stock photo
- Engagement bait ("Like хийгээгүй бол алгана" гэх мэт)
- Caption хуулсан сонирхолтой пост (өмнө хэн нэгэн бичсэн)

## 📐 7-хоногийн ритм (контент календарь)

```
Дав   Reel 30сек — Demo flythrough         (velocity өндөр)
Мяг   Carousel 7 слайд — "Excel алдаа"     (saves өндөр)
Лха   Text-post — Case study (testimonial) (comments өндөр)
Пүр   Native фото — Behind the scenes      (humanize)
Баа   Reel 60сек — Founder ярилцлага       (trust)
Бям   Light meme/quote                     (reach)
Ням   Q&A пост (асуултаар хариу татна)     (comment-velocity)
```

Энэ ритм нь алгоритмд "тогтмол креатор" гэсэн сигнал илгээж, account-ийн дундаж reach-ийг аажмаар нэмэгдүүлнэ.
