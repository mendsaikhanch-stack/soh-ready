## 🎯 4 үе шаттай Content Mapping

### 🟦 1. AWARENESS — "Анхаарал татах" (Reach 70%)

**Зорилго:** Скролл зогсоох, асуудлыг таниулах, дагалт бий болгох

| Контент төрөл | Format | Тоо хэмжээ | Жишээ |
|---------------|--------|-----------|-------|
| **Pain-point Reel** | 30сек босоо видео | 4/сар | "СӨХ-ний 5 нийтлэг алдаа" |
| **Statistic carousel** | 7 слайд | 2/сар | "Монголын СӨХ-ний тоо: 12,000+, дижиталжсан нь: 4%" |
| **Polarizing quote** | Текстэн пост | 4/сар | "СӨХ-ний тэргүүн = сайн дурын ажилтан биш, менежер" |
| **Meme/Хошин** | Зураг | 4/сар | СӨХ-ний бүлгийн чатын утга төгөлдөр screenshot |
| **Q&A asking** | Текст | 2/сар | "Танай байр СӨХ-тэй юу? Хэрхэн ажилладаг вэ?" |

**KPI:** Reach, Saves, Shares
**Үе шатнаас гаргах сигнал:** Comment → DM рүү татна

---

### 🟩 2. VALUE — "Үнэ цэн өгөх" (Reach 50%)

**Зорилго:** Боловсрол өгөх, шийдлийн ангиллыг тайлбарлах, өөрийгөө шинжээч гэдгээ батлах

| Контент төрөл | Format | Тоо хэмжээ | Жишээ |
|---------------|--------|-----------|-------|
| **How-to tutorial** | Reel 60сек | 4/сар | "Хэрхэн СӨХ-ний жилийн төсөв гаргах вэ — 5 алхам" |
| **Comparison table** | Carousel | 2/сар | "Excel vs Apps vs ХОТОЛ — Хэн юугаараа сайн вэ" |
| **Tip series** | Text + зураг | 4/сар | "СӨХ-ний тэргүүнд: цуглааны ирц 80% болгох 3 арга" |
| **Industry insight** | Text-post | 2/сар | "Дэлхийн HOA software market — Монголын боломж" |
| **Live workshop** | FB Live 30 мин | 1/сар | "СӨХ-ний дижитал шилжилт — A to Z" |
| **Free resource (PDF)** | Post + DM | 2/сар | "СӨХ-ний жилийн төсвийн загвар PDF татах" |

**KPI:** Saves, Mid-funnel DM
**Үе шатнаас гаргах сигнал:** PDF татсан → Email list-д орох → Trust үе шатанд

---

### 🟨 3. TRUST — "Итгэл бүтээх" (Reach 30%)

**Зорилго:** Бид ажилладаг гэдгээ нотлох, эргэлзээг арилгах, нийгмийн нотолгоо

| Контент төрөл | Format | Тоо хэмжээ | Жишээ |
|---------------|--------|-----------|-------|
| **Case study** | Carousel 7 слайд | 2/сар | "X хороолол ХОТОЛ ашигласнаар цуглуулалт 45% нэмэгдсэн" |
| **Customer testimonial** | Reel 60сек | 2/сар | СӨХ-ний тэргүүний ярилцлага |
| **Behind-the-scenes** | Native фото | 2/сар | Хөгжүүлэгчдийн баг, дэлгэцэн дээрх мэдээлэл |
| **Numbers post** | Зураг + текст | 1/сар | "1 жилд: 10,000 оршин суугч, 50 хороолол, ₮300сая ил тод гүйлгээ" |
| **Founder story** | Text + фото | 1/сар | "Бид яагаад ХОТОЛ-ыг бүтээх болов?" |
| **FAQ post** | Carousel | 1/сар | "Хамгийн их асуудаг 5 асуулт" |
| **Partnership announce** | Зураг + текст | 1/сар | "Банктай хамтын ажиллагаа" |
| **Press mention** | Repost | Тохиолдсон үед | "Eagle.mn-д тухай нийтлэл" |

**KPI:** Comments, Profile visits, Demo booking page hits
**Үе шатнаас гаргах сигнал:** "Энэ хороолоход байгаа юу?" гэх асуулт → Conversion

---

### 🟥 4. CONVERSION — "Хөрвүүлэх" (Reach 15%)

**Зорилго:** Худалдан авах үйлдэлд хүргэх, Demo захиалга, гэрээ

| Контент төрөл | Format | Тоо хэмжээ | Жишээ |
|---------------|--------|-----------|-------|
| **Demo invite** | Reel + link | 2/сар | "Үнэгүй 15 минут демо — Calendly link" |
| **Limited offer** | Зураг + текст | 1/сар | "Эхний 100 СӨХ-нд эхний 3 сар үнэгүй" |
| **Direct CTA post** | Текст + ad | 4/сар | "Танай хорооллыг ХОТОЛ дээр оруулах 3 алхам" |
| **Objection-busting** | Reel | 1/сар | "Өндөр настай оршин суугч аппыг хэрхэн ашиглах вэ" |
| **Urgency/FOMO** | Текст | 2/сар | "Энэ долоо хоногт 3 шинэ хороолол нэгдэв" |
| **Risk reversal** | Текст | 1/сар | "30 хоногийн дотор хангалтгүй бол мөнгө буцаана" |

**KPI:** Demo booked, Trial sign-ups, Contract signed
**Revenue trigger:** Demo → Onboarding call → Pilot гэрээ → Жилийн гэрээ

## 🔄 Системийн "Flow" — Постууд хоорондоо хэрхэн холбогдох вэ

```
Awareness Reel
   │ (40% scroll past, 60% хариу үйлдэл хийнэ)
   ▼
Comment: "Манайд иймэрхүү байна!"
   │
   ├─► DM auto-reply: "Free PDF татаарай" → Value content рүү
   │
   ▼
Free PDF татсан (Email captured)
   │
   ▼
Email seq: 3 өдрийн дараа Value Tutorial
   │
   ▼
Case study post харж сэтгэгдэл бичсэн
   │
   ▼
Retargeted ad: "Demo захиалах" (Conversion)
   │
   ▼
Demo booked → Sales call → Гэрээ
```

## 💰 Revenue Attribution (ямар контент мөнгө авчирдаг вэ)

| Үе шат | Reach × Conversion | Revenue per post (тооцоо) |
|--------|---------------------|----------------------------|
| Awareness | 10,000 × 0.5% → Value | ₮ 0 |
| Value | 5,000 × 5% → Trust | ₮ 0 |
| Trust | 1,500 × 8% → Conversion | ₮ 800,000 *(дам)* |
| Conversion | 120 × 15% → Sale | ₮ 5,400,000 *(шууд)* |

**Гол ойлголт:** Conversion пост хийхээс өмнө **Awareness × Value × Trust** гэсэн 3 үе шат хийгээгүй бол → үр дүн 80%+ буурна.
