# Хотол Android — Тестийн checklist

Release `.aab` гаргахаас өмнө (debug APK эсвэл internal testing build дээр) бүгдийг шалга.

## Суулгалт ба харагдац
- [ ] 1. App амжилттай суулгана (debug APK / internal test).
- [ ] 2. App icon (барилгын тэмдэг) launcher дээр зөв харагдана.
- [ ] 3. App нэр "Хотол" зөв харагдана.
- [ ] 4. Splash screen (Хотол лого, цагаан дэвсгэр) зөв ажиллана, ~1.5с дараа арилна.
- [ ] 5. Status bar өнгө (#2563eb) brand-тай зохицсон.

## Үндсэн ачаалалт
- [ ] 6. App нээгдэхэд `https://www.khotol.com` зөв ачаална.
- [ ] 7. Эхний ачаалалт удаан үед хоосон цагаан биш — splash/loading харагдана.
- [ ] 8. Нэвтрэх (login) хуудас ажиллана.
- [ ] 9. Demo account (Утас 88000000 / Нууц үг Demo12345!) нэвтэрнэ.
- [ ] 10. Dashboard ачаалж, контент харагдана.

## Навигаци
- [ ] 11. Back button: веб түүх байвал өмнөх хуудас руу буцна.
- [ ] 12. Back button: түүх дуусвал app-аас гарна (хоосон дэлгэц үлдээхгүй).
- [ ] 13. Гадны линк (email `mailto:`, утас `tel:`) зөв нээгдэнэ.

## Play-д шаардлагатай хуудаснууд (app дотроос нээгдэх ёстой)
- [ ] 14. Privacy Policy — `/privacy` нээгдэнэ.
- [ ] 15. Terms — `/terms/resident`, `/terms/admin` нээгдэнэ.
- [ ] 16. Account deletion — `/account/delete` нээгдэж, форм ажиллана.
- [ ] 17. Contact / demo авах — `/contact`, `/demo-request` ажиллана.
- [ ] 18. Privacy/Terms дээрх `[он/сар/өдөр]` placeholder бодит огноогоор солигдсон.

## Функцийн тест
- [ ] 19. Төлбөрийн **мэдээлэл харах** ажиллана.
- [ ] 20. Зар мэдээлэл харагдана.
- [ ] 21. Хүсэлт/гомдол илгээх ажиллана.
- [ ] 22. Тайлан харагдана.
- [ ] 23. File upload (зураг/баримт) — gallery-аас сонгох ажиллана.
      ⚠️ Камераар шууд авах нь нэмэлт тохиргоо шаардаж болзошгүй — тусд нь тест хий.

## Сүлжээ / алдаа
- [ ] 24. Интернэт **унтрааж** app нээхэд → offline.html ("Интернэт холболт алга") харагдана.
- [ ] 25. Offline дэлгэцийн "Дахин оролдох" товч → онлайн болоход вебийг ачаална.
- [ ] 26. Dark mode дээр контент уншигдахуйц (хар дэлгэц + хар текст биш).

## Build / store
- [ ] 27. targetSdk 36 (≥35) — Play-ийн шаардлага хангасан.
- [ ] 28. Permissions = зөвхөн INTERNET (илүү permission асуухгүй).
- [ ] 29. Release `.aab` амжилттай гарна (`app-release.aab`).
- [ ] 30. `.aab` нь release keystore-оор гарын үсэг зурагдсан (debug биш).
- [ ] 31. Internal testing-д upload амжилттай, төхөөрөмж дээр суулгаж туршсан.
