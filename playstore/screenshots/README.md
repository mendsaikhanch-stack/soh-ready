# Хотол — Google Play screenshots

Production (`www.khotol.com`)-оос demo бүртгэлээр (утас `88000000`) автоматаар авсан.
Бүгд **1236×2196 px (9:16)** — Play-ийн phone screenshot шаардлага хангана
(тал бүр 320–3840px, харьцаа 16:9–9:16).

Дахин авах:
```bash
node scripts/capture-store-screenshots.mjs   # → scripts/screenshots/*.png
```

## Анхаарах
- Бүх дэлгэц **demo дата** — бодит хүний/СӨХ-ийн мэдээлэлгүй
  ("Та (демо хэрэглэгч)", "Туршилтын СӨХ — Хотол демо байр", утас `77000000`/`88000000`).
- **Орхигдуулсан** дэлгэцүүд: payments (demo-д 0₮ өргүй тул хоосон),
  reports (энэ сард тайлан ороогүй), guide (QPay "төлбөр төлөх"-ийг
  сурталчилдаг — production-д QPay идэвхгүй тул store-д тавихгүй).
- Илүү дэлгүүний харагдац хүсвэл demo дата баяжуулаад дахин авна.
